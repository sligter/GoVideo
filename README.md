# GoVideo
视频资源采集站聚合播放器

软件为空壳，需导入源后使用

设置 -> 订阅源管理 -> 添加订阅源 -> + -> 输入链接 -> 确定 ->更新

V1.0.0 
GoVideo更新日志:

- 1.支持同时搜索多个源
- 2.支持在线搜索并加载弹幕
- 3.支持下载视频
- 4.边下边播
- 5.订阅源
